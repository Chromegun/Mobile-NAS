# 📱 Mobile NAS Project 
**안 쓰는 안드로이드 스마트폰을 음악 작업용 NAS 서버로 변신시키기**

본 프로젝트는 루팅된 안드로이드 기기(Linux Deploy 환경)를 활용하여, 저전력 고효율의 파일 공유 서버를 구축할 수 있게 도와줍니다.

## ✨ 주요 기능
- **Samba 파일 공유**: 윈도우 탐색기에서 네트워크 드라이브로 즉시 연결
- **Tailscale 지원**: 외부 어디서든 집 안의 서버에 원격 접속
- **한글 대시보드**: 깔끔한 터미널 UI와 실시간 서비스 진단 기능

## 🚀 설치 방법
터미널에서 아래 명령어를 한 줄만 입력하세요:
```
bash
curl -sSL https://raw.githubusercontent.com/Chromegun/Mobile-NAS/main/setup.sh | bash
```
